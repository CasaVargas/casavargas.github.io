How we build

Native or nothing

Every line of code is SwiftUI. Every pixel is intentional. Every app is built to feel like it belongs on your device.

Spec-first development

Every feature starts as a detailed specification before a single line of code is written. The thinking happens upfront so the building flows.

Pure SwiftUI

No web views. No React Native. No compromise. Native frameworks deliver the performance and feel that Apple users expect.

Ship with polish

The details matter — from smooth animations to thoughtful empty states. An app isn't done when it works, it's done when it feels right.

About

One developer,
many ideas

CasaVargas is a solo indie studio founded by Jon Vargas, based in North Carolina. What started as a passion for building things that feel right has grown into projects spanning Apple platforms and the desktop — from OneScribe on iOS to DebridDownloader across macOS, Windows, and Linux.

When not writing SwiftUI or Rust, Jon works in K-12 IT, tinkers with self-hosted infrastructure, and is always dreaming up the next thing to build.
